# App Store 模擬

## 簡介

使用 Vite 和 React，建立模擬 App Store 應用程式的頁面，功能包括推薦應用程式列表、前 100 名應用程式列表，和搜尋應用程式功能。

## 使用指南

```bash
git clone https://github.com/PTPE/app-store.git
cd app-store
```

### 安裝套件

```bash
npm install
# 或
yarn install
```

### 起專案

```bash
npm run dev
# 或
yarn run dev
```

## 使用技術

- **Vite**
- **React**
- **Tailwind CSS**
- **React Query**
